# ft_printf

In this Hive Helsinki school project I recoded the libc's printf function.

## Conversions and flags handled

- `c s p`
- `d i o u x X` with `hh, h, l, ll`
- `f` with `l, L`
- `%%`
- flags `# 0 - +`
- minimum width
- precision
- `\*`

## Bonuses outside of printf

- print in colors: `{color name} content {eoc}`
- binary conversion `b`
